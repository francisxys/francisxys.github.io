简介

MyCli 是一个 MySQL 命令行工具，支持自动补全和语法高亮。也可用于 MariaDB 和 Percona。
官网介绍
Mycli is tested on macOS and Linux. It runs on Python 2.7 and 3.4+. Works well with unicode input/output. NOTE: Python 2.6 support was dropped in mycli 1.9.0. If you're running Python 2.6, you'll want to install mycli 1.8.1.
Mycli就是说支持Python 2.7和Python 3.4+版本。
如果系统Python版本是2.6，那么安装后的Mycli 版本为1.8.1。
